
# mvs-aws-bachelorthesis

Dieses Repository beinhaltet Skripte und Konfigurationsdateien zur Erstellung, Konfiguration und Bereinigung einer Übungs- und Versuchs­umgebung für praktische Untersuchungen im Bereich Cloud-basiertes Video-Transcoding und Content-Auslieferung. 

Der Fokus liegt auf der Nutzung von Object-Storage- und CDN-Diensten der Anbieter :contentReference[oaicite:0]{index=0} und :contentReference[oaicite:1]{index=1}. Ergänzend dazu enthält das Repository die zugehörige technische Dokumentation des Workflows sowie der eingesetzten Komponenten.

Das Repository entsteht im Rahmen der Bachelorarbeit im Studiengang Medientechnik an der :contentReference[oaicite:2]{index=2} und dient der strukturierten Umsetzung sowie Dokumentation der experimentellen Anteile der Arbeit.
